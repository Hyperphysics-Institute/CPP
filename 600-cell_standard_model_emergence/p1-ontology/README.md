# Minimalist Ontology and Consciousness in Conscious Point Physics (CPP)

This directory documents the foundational ontological principles from Paper 1 (Sections 1.2–1.3): the universe as emergent from interactions of Conscious Points (CPs) as distributed processors within the 600-cell lattice. Consciousness is primitive—not emergent—enabling rule enforcement via DI-bits (informational currency). The framework synthesizes historical ideas from Dirac, Gell-Mann, Conway, Bohm, Penrose, and Wheeler into a participatory universe model.

Key concepts:
- CPs as rule-enforcing processors
- DI-bits as informational currency
- Emergent complexity from minimalist ontology
- Historical context and influences

Cross-references: Paper 1 Sections 1.1 (Historical Context), 1.2 (Minimalist Ontology), 1.3 (Consciousness as Foundational Mechanism).
