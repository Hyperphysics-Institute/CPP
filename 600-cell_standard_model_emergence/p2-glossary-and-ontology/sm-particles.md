# Standard Model Particles in CPP

Descriptions of SM particles in terms of CPP structures:

- **Electron**  
  Central unpaired -eCP, polarized DP cloud (~25% uniform mix), single orbital eDP for spin. Minimal cage (N_k=1). Mass from base polarization + orbital ZBW + cloud.

- **Muon**  
  Central -eCP, tetrahedral cage (N_k=4), orbital eDP with fractional qDP/hDP mixing. Mass from base + orbital + inter-layer bonding + cloud.

- **Tau**  
  Central -eCP, icosahedral cage (N_k=12) around mu-tetra. Orbital eDP with mixing. Highest lepton mass from multi-layer bonding.

- **Up quark**  
  Central +qCP, bare/minimal structure, radial qDP-favored cloud. Orbital qDP-dominant ZBW. +2/3 charge from single screening.

- **Down quark**  
  Central -qCP, linear qDP/hDP extra (Capotauro bias). Orbital ZBW + linear extra. -1/3 charge from double screening.

- **Strange, Charm, Bottom, Top quarks**  
  Increasing nested cages (tetra → icosa → dodeca → fullerene-like). Radial qDP gradients, linear extras (down-type). Mass from multi-layer bonding + ZBW contributions.

- **Neutrinos (ν_e, ν_μ, ν_τ)**  
  No central CP. Unbound orbital ZBW: eDP (ν_e), qDP (ν_μ), hDP-tetra (ν_τ). Maximal σ = 120^{-3} suppression → tiny masses.

- **W boson**  
  Linear chain of 6 hDPs (ribbon loop). No central CP. Mass from linear ZBW oscillations. W± forms via bond with ±CP in high-energy conditions.

- **Z boson**  
  Icosahedral cage (180 effective vertices), 25% DP mix. Symmetric, closed structure → neutral, clean decays.

- **Higgs boson**  
  Dodecahedral cage (3000 effective), 25% mix. Diffuse cloud → highest boson mass.
