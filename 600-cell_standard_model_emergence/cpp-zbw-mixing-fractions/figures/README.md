## Figures and Visualizations

The figures in the `figures/` directory illustrate the orbital Zitterbewegung (ZBW) DP mixing fractions predicted by the Conscious Point Physics (CPP) model. These visuals support the preregistered prediction for the muon g-2 anomaly correction (δμ ≈ (2.0–3.0) × 10⁻⁷) by demonstrating how fractional mixing of eDP, qDP, and hDP in orbital ZBW arises from energy minimization and thermal equilibrium in the 600-cell lattice cages.

All figures are generated automatically by running the notebooks in the `notebooks/` folder or by using the `mixing_calculator.py` module in `src/`.

### lepton_pie_chart.png
**File**: `figures/lepton_pie_chart.png`  
**Description**: Pie chart showing the equilibrium orbital ZBW DP composition for a muon-like lepton (N_k = 4 tetrahedral cage).  
**Key result**: ~68.5% eDP (dominant), ~13% qDP, ~18.5% hDP.  
**Interpretation**: In leptons, eDP is energetically favored due to lower SSV gradients, but cage complexity (N_k > 1) introduces non-negligible qDP/hDP mixing. This fractional admixture is the geometric origin of the predicted muon g-2 deviation in the model.  
**Generated by**: `notebooks/lepton_zbw_mixing.ipynb`

### quark_pie_chart.png
**File**: `figures/quark_pie_chart.png`  
**Description**: Pie chart showing the orbital ZBW DP composition for a representative light quark (e.g., up quark, N_k = 1 minimal cage).  
**Key result**: ~74% qDP (strongly dominant), with smaller contributions from hDP and eDP.  
**Interpretation**: Quarks exhibit reversed gradient ordering compared to leptons, making qDP the lowest-energy state. This strong qDP preference is consistent across generations but weakens slightly in heavier quarks with larger cages.  
**Generated by**: `notebooks/quark_zbw_mixing.ipynb` (single N_k=1 calculation)

### sensitivity_plot.png
**File**: `figures/sensitivity_plot.png`  
**Description**: Line plot showing how the dominant DP fraction (qDP for quarks) evolves with cage occupancy N_k across quark generations (from up quark N_k=1 to top quark N_k≈30,000).  
**Key axes**:  
- X-axis: N_k (log scale)  
- Y-axis: qDP fraction in orbital ZBW  
**Interpretation**: At small N_k (light quarks), qDP dominance is extreme (~97–99%). As N_k increases (heavier generations), thermal equalization slightly reduces the fraction, reflecting the transition toward more uniform mixing in very large cages. This N_k dependence underpins generational mass hierarchies and helps constrain the mixing fractions used in the g-2 prediction.  
**Generated by**: `notebooks/quark_zbw_mixing.ipynb` (sensitivity sweep section)

### Reproducing the Figures
1. Run `notebooks/lepton_zbw_mixing.ipynb` → produces `lepton_pie_chart.png`
2. Run `notebooks/quark_zbw_mixing.ipynb` → produces `quark_pie_chart.png` and `sensitivity_plot.png`
3. Alternatively, use the command-line interface in `src/mixing_calculator.py`:
   ```bash
   python src/mixing_calculator.py --type lepton --N_k 4    # prints lepton fractions
   python src/mixing_calculator.py --type quark --N_k 1     # prints up quark fractions
   python src/mixing_calculator.py --type quark --sweep     # runs N_k sweep (add plotting if desired)
