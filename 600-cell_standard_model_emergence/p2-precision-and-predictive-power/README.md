# Precision and Predictive Power of Conscious Point Physics (CPP)

This directory documents the precision achievements of the CPP framework as presented in Paper 2 (Version 29), Appendix N. It highlights:
- 100% agreement with PDG central values for 18+ Standard Model particle masses using **one single calibration** (electron mass → k ≈ 0.0185)
- Correct order-of-magnitude post-diction of the resolved muon g-2 residual (~2.9 × 10^{-10}, within 1σ of the 2025 Fermilab + lattice QCD result)
- Current status and roadmap for other fundamental constants (α_em, sin²θ_W, neutrino angles, G, Λ)

The framework derives these results from the 600-cell lattice geometry (120 vertices, golden ratio φ ≈ 1.618, holographic suppression σ = 120^{-d}) without per-particle fitting parameters.

Cross-references: Paper 2 Appendix N (Precision and Predictive Power), Section 6 (mass tables), Appendix B.1 (g-2 mixing).
