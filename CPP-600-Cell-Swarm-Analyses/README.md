## File Organization Standards

Each trial subdirectory should contain:

## Independence Protocol

### Cross-Trial Contamination Prevention
- **Phenomenon Exclusion**: No overlap between trial datasets
- **Methodological Consistency**: Same classification criteria across trials
- **Blind Selection**: Selector isolated from previous trial results during phenomenon identification

### Quality Assurance
- **Complete Documentation**: Full audit trail for each trial
- **Reproducible Analysis**: Standardized statistical methods
- **Transparent Reporting**: All results published regardless of outcome
