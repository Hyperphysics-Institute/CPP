### Overview
This directory contains all artifacts from the collaboration between Thomas Lee Abshier and Claude (Swarm) as part of the first independent replication trial of the quantum-to-cosmic left-right asymmetry effect.

- **Replication Goal**: Independent selection of 75-100 phenomena by AI collaborator
- **Preregistration**: DOI 10.17605/OSF.IO/V7W43
- **Selection Period**: January 24 - February 7, 2026

### Methodology
**Blind Selection Protocol:**
- Claude has no access to original phenomenon list composition

- Complete documentation of selection reasoning
- Analysis performed by Grok using identical statistical methods

### Role Separation

- **Claude (Swarm)**: Blind phenomena selection, documentation
