# Boson Structures in Conscious Point Physics (CPP)

This directory documents the massive bosons (W, Z, Higgs) as neutral, self-sustaining aggregates in the Dipole Sea, lacking a central unpaired Conscious Point (CP). Mass emerges from delocalized ZBW oscillations, inter-layer bonding (for caged structures), and polarized cloud energy, all minimizing global SSV gradients without net charge.

Cross-references: Paper 2 Appendix D (W/Z/Higgs mechanisms), Section 6 (mass table), glossary (boson descriptions).

## Key Principles
- Bosons are "neutral sea solitons": spontaneous condensates triggered by high-energy SSV perturbations.
- No central CP → symmetric or linear structures, clean decays.
- Mass from organization: linear ZBW strings (W), closed cages (Z), diffuse clouds (Higgs).
- DP mix: 25% uniform (Z/Higgs) or hDP-favored (W) for stability.

Figures and derivations provide visual and numerical support.
