# Structure Formation from Lattice Defects

Galaxies and large-scale structure are seeded by lattice defects from Capotauro.

## Mechanism
- Incomplete Capotauro breaking → stable topological defects
- Defects have mass ~GeV–TeV, interact weakly via residual SSV
- Seeds density perturbations → gravitational collapse → galaxies
