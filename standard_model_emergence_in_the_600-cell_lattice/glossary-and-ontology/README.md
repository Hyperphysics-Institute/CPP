# Glossary and Ontology in Conscious Point Physics (CPP)

This directory provides a complete, self-contained glossary of terms used across the CPP framework, with emphasis on their ontological significance. Consciousness is not an emergent property but a primitive: each Conscious Point (CP) is a distributed processor that enforces universal rules via DI-bits (informational currency). This panpsychist foundation unifies physics and observation—CPs "witness" interactions holographically, enabling symmetry breaking and emergent phenomena without external tuning.

Terms are grouped as in Paper 2 glossary: particle factors, SM particles, geometric/lattice terms, ZBW/spin terms, Dipole Sea/organization terms, and miscellaneous.

Cross-references: Paper 2 Glossary section, Appendix on CP processors and consciousness role.
